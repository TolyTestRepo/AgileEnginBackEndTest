### Prerequisites
* Java 1.8 or higher

## Getting Started
To build appliction
```
mvn clean install  
```
To run application 
```
java -jar HTMLAnalyzer-1.0-SNAPSHOT.jar <input_origin_file_path> <input_other_sample_file_path> <id_of_search_element>
```
By default appliction analyze and search element with id = "make-everything-ok-button"
